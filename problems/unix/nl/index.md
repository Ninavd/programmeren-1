# UNIX: nl

Schrijf een tool genaamd `nl` die de inhoud van een file naar de terminal print, met regelnummers vóór elke regel (tel vanaf 1). De naam van de file wordt opgegeven als command-line argument.

    $ ./nl woorden.txt
         1 aback
         2 burnt
         3 coyly
         4 drawn
         5 extra
         6 flirt
         7 graph
         8 haute
         9 imply
        10 jumpy
    ...
